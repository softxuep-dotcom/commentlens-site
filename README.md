# CommentLens Website

Static website for Creem production review.

## Files

- `index.html` - product landing page
- `privacy.html` - privacy policy
- `terms.html` - terms of service
- `styles.css` - shared styling

## Deploy without a server

Use Cloudflare Pages or GitHub Pages. Both can host static files for free. For Creem review,
bind a custom domain such as `https://commentlens.yourdomain.com`; do not submit a platform
subdomain such as `*.pages.dev`, `*.vercel.app`, or `*.netlify.app`.

## Before submitting to Creem

- Bind a custom domain.
- Confirm `https://your-domain/`, `/privacy.html`, and `/terms.html` load publicly.
- Replace any support wording if you have a real support email or Chrome Web Store listing URL.
