Privacy Policy

Last updated: June 5, 2026

Overview

CommentLens is a free Chrome extension for collecting, exporting, and analyzing comments from YouTube, Reddit, and X (Twitter) pages opened by the user. Comment data stays on the user's device unless the user chooses to export it.

Data handled by the extension

When the user starts a collection, CommentLens reads the visible comments from the active YouTube, Reddit, or X (Twitter) tab. This may include comment text, author display names, like or score counts, reply counts, timestamps, nesting depth, and comment links. This data is stored locally in the browser using Chrome extension storage and can be exported by the user as a CSV file.

No external network requests

CommentLens does not send your data anywhere. It has no accounts, no login, and no license system, and it makes no external network requests. All collection, analysis, and export happen entirely on your device.

No analytics or tracking

CommentLens does not include third-party analytics, advertising trackers, or behavior tracking scripts.

Local storage

Collected comments and popup preferences are stored in Chrome local extension storage. Users can remove this data at any time using the "Clear data" button in the extension, or by uninstalling the extension.

Data sharing

CommentLens does not sell, transfer, or share user data with any third party. No data ever leaves the user's device.

Contact

For privacy or support requests, use the support channel provided in the Chrome Web Store listing.
