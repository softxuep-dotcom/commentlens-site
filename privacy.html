Privacy Policy

Last updated: May 29, 2026

Overview

CommentLens is a Chrome extension for collecting, exporting, and analyzing comments from YouTube pages opened by the user. The extension is designed to keep comment data on the user's device unless the user chooses to export it.

Data handled by the extension

When the user starts a collection, CommentLens reads visible YouTube comment data from the active YouTube tab. This may include comment text, author display names, like counts, reply counts, timestamps, and comment links. This data is stored locally in the browser using Chrome extension storage and can be exported by the user as a CSV file.

License verification

If a user activates a Pro license, the extension sends the license key and license instance information to a Cloudflare Worker operated for CommentLens. The Worker forwards the license verification request to Creem. This is used only to activate, validate, or deactivate paid access.

No analytics or tracking

CommentLens does not include third-party analytics, advertising trackers, or behavior tracking scripts inside the extension.

Local storage

Collected comments, license status, license key, license instance id, and daily usage count may be stored in Chrome local extension storage. Users can remove extension data by clearing the extension's site data or uninstalling the extension.

Data sharing

CommentLens does not sell personal data. License verification data is shared with Creem only as necessary to manage paid license access.

Contact

For privacy or support requests, use the support channel provided in the Chrome Web Store listing or the purchase receipt.
